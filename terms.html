9 Service Description & MTD Compliance

9.1 What VATflow Provides

VATflow provides software that enables VAT-registered businesses to:

  • Maintain digital VAT records in compliance with SI 2018/261
  • Prepare 9-box VAT returns for all VAT accounting schemes
  • Submit VAT returns to HMRC via the MTD VAT API
  • Retrieve obligations, liabilities, and payment information from HMRC
  • Import transaction data from accounting software (Xero, Sage, FreeAgent, QuickBooks, Anna Money)

9.2 MTD Compliance Statement

VATflow satisfies all four HMRC Making Tax Digital requirements:

  1. Keeping digital records of VAT transactions
  2. Maintaining digital links throughout the data journey
  3. Preparing the VAT return from digital records
  4. Submitting the return directly to HMRC via the MTD API

⚠️ VATflow is a software tool. It is not a tax advisory service. We do not provide tax advice. You remain responsible for ensuring your VAT figures are accurate and that your returns are filed correctly and on time.
