VAT Accounting Schemes
Select your scheme — Box 1 and Box 4 calculations update automatically across all 9 boxes
Active Scheme
Standard VAT (Invoice Basis)
Box 1 = sum of VAT on all sales invoices. Box 4 = sum of VAT on all purchases. For businesses above £1.35m turnover or choosing standard. Full input VAT recovery.
Cash Accounting Scheme
VAT accounted when payment received/made — not invoice date. Eligible if turnover ≤ £1.35m. Improves cash flow. Box 4 = VAT on paid purchases only.
Flat Rate Scheme (FRS)
Box 1 = Gross sales (incl. VAT) × FRS %. Box 4 = £0. Exception: capital goods ≥ £2,000 incl. VAT. Eligible if turnover ≤ £150,000. Simpler administration.
Annual Accounting Scheme
One return per year, advance payments during the year. Eligible if turnover ≤ £1.35m. Reduces admin to single filing. Same 9-box format covering full year.
Retail Scheme
For retailers with mixed VAT rates where recording per item is impractical. Uses Daily Gross Takings × applicable fraction. Standard Box 4 input recovery.

UK VAT Law & Regulations
Current legislation — 2025 · Reference only, not legal advice
Value Added Tax Act 1994 (VATA 1994) VATA 1994
Principal UK VAT statute. Covers registration, charging, recovery and payment. Key sections: s.1 (imposition), s.4 (scope of VAT), s.24–26 (input tax recovery), s.69–70 (penalties), Schedule 1 (registration threshold £90,000 from 1 Apr 2024), Schedule 8 (zero-rated supplies), Schedule 9 (exempt supplies).
Finance Act 2021 — MTD Extended to All VAT-Registered Businesses FA 2021
From 1 April 2022, ALL VAT-registered businesses must use MTD-compatible software — regardless of turnover. Previously only those above the £85,000 threshold (from April 2019). VATflow satisfies all four core MTD requirements.
MTD for VAT Regulations 2018 — Digital Records & Links SI 2018/261
Requires: (1) Keep digital records — business name, address, VRN, scheme. (2) Record every supply: time, value ex. VAT, VAT rate, VAT amount. (3) Prepare 9-box return from digital records. (4) Submit via API. Digital links required throughout. Permitted: CSV, API, formula-linked cells. Prohibited: manual retyping, copy-paste between systems.
VAT Registration Threshold 2024–2026 Notice 700/1
From 1 April 2024: registration threshold = £90,000 annual taxable turnover; deregistration threshold = £88,000. Voluntary registration permitted below threshold. Both thresholds frozen until April 2026 (Spring Budget 2024).
Post-Brexit VAT — TCTA 2018 & Windsor Framework TCTA 2018
Goods to/from EU are now exports/imports (not EC acquisitions). Postponed VAT Accounting (PVA) available for import VAT. Northern Ireland Windsor Framework 2023: EU VAT rules still apply to goods (not services) in NI — affects Boxes 2, 8, and 9. Box 2 = VAT on NI EC acquisitions only.
Points-Based Penalty Regime (from 1 January 2023)
Replaced the default surcharge. Late submission: 1 point per late return; £200 fine when threshold reached (4 points for quarterly filers). Late payment: 2% penalty at 16 days, 4% at 31 days, plus daily interest at BoE base + 2.5%. Failure to maintain digital records: up to £3,000 per failure. Failure to maintain digital links: up to £3,000 per failure.
Flat Rate Scheme — HMRC VAT Notice 733
FRS eligible if taxable turnover ≤ £150,000. Box 1 = Gross sales (net + VAT) × FRS percentage. No input VAT recovery (Box 4 = £0) except: single capital goods purchase ≥ £2,000 incl. VAT at time of registration. Limited Cost Trader rules apply if goods < 2% of turnover (flat rate = 16.5%). Leave FRS when turnover exceeds £230,000 for 12 months.

Step 1 — Create Account & Add Company

Sign up at the login screen. Plans from £2/month per company — Monthly, Quarterly (save 17%), or Annual (save 38%). 14-day free trial, no card required. Add additional companies from the top-right user menu at £2/month each.

Step 2 — Connect to HMRC Government Gateway

Click Authorise with HMRC Government Gateway in the header. Enter your VRN. Choose Sandbox (testing) or Production. Clicking Authorise opens HMRC's own Government Gateway in a new tab — you log in there and grant VATflow permission. VATflow stores only an encrypted OAuth access token — never your password. Once authorised, open obligations are pulled automatically.

Step 3 — Enter Records or Upload Template

Options: (a) add transactions manually in Digital Records — all buttons fully functional, (b) import a CSV/Excel file, (c) connect accounting software via Integrations (Xero, Sage, FreeAgent, QuickBooks, Anna Money), or (d) download the Excel Template, fill in the Sales and Expenses tabs, then upload it back. All routes maintain the digital link required by HMRC MTD.

Step 4 — File Return, Validate & Pay

Go to File a Return → Auto-populate from Records → Validate → Submit to HMRC. A confirmation reference is returned immediately from HMRC's API. Then go to Payments and pay by card, Direct Debit, bank transfer, or Open Banking — all directly within the app.

  1. Download from Excel Template page

Can I correct errors after submission?
Errors ≤ £10,000 (or 1% of Box 6, whichever is higher): correct on the next VAT return. Larger errors: disclose to HMRC using form VAT652. Voluntary disclosure reduces penalties. Correcting proactively is always better than HMRC discovering an error.
How long must I keep VAT records?
6 years minimum (10 years for VAT MOSS). VATflow stores all digital records and allows export at any time. Your Excel template file also constitutes your digital record under MTD regulations — keep it for 6 years.
Can I use my own Excel spreadsheet (not this template)?
Yes — use Digital Records → Import CSV/Excel to upload your own file, or use Integrations to link cells via bridging software mode. VATflow maintains the digital link from your spreadsheet to HMRC. Do not manually retype any values.
What if the HMRC connection fails?
Try Sandbox mode first to verify your credentials work. For Production, ensure your HMRC Developer Hub application has the correct scopes (read:vat write:vat) and production credentials have been approved. OAuth tokens expire after 4 hours — VATflow auto-refreshes using the refresh token.
Can multiple users access one company account?
Multi-user with role-based access (Admin / Preparer / Viewer) is on the product roadmap. Currently one login per company. Tax agents can act on behalf of clients using HMRC's Agent Authorisation service.
HMRC Approval Process
Steps to achieve HMRC recognition as MTD-compatible software
ℹ HMRC does not issue a formal "stamp of approval". Instead: register on the HMRC Developer Hub → complete testing → satisfy all technical requirements → get listed on HMRC's compatible software register at gov.uk.
Required Steps (complete in order)
1
Register on HMRC Developer Hub
Go to developer.service.hmrc.gov.uk → Sign In with Government Gateway → Applications → Add Application → Select "VAT (MTD)" API → Choose Sandbox → Obtain Client ID and Client Secret.
2
Complete Sandbox Testing
Test all five API endpoints using HMRC's test users. Required: retrieve obligations (GET), submit return (POST), retrieve submitted return (GET), view liabilities (GET), view payments (GET). All must return HTTP 200 with correct JSON.
3
Implement Fraud Prevention Headers (MANDATORY)
Every API call must include: Gov-Client-Connection-Method, Gov-Client-Device-ID, Gov-Client-User-IDs, Gov-Client-Timezone, Gov-Client-User-Agent, Gov-Client-Public-IP, Gov-Vendor-Version, Gov-Vendor-Product-Name. Failure = API rejection.
4
Apply for Production Credentials
Developer Hub → Your Application → Request Production Access. HMRC reviews: correct OAuth scopes, Terms of Use accepted, fraud prevention headers validated, Privacy Policy URL provided, accessibility statement provided.
5
Legal & Compliance Requirements
Before production approval: Privacy Policy (UK GDPR), Terms & Conditions, ICO Data Controller registration, SSL/TLS on all endpoints, annual penetration test, Data Processing Agreement template for clients.
6
Submit to HMRC Compatible Software Register
Once production credentials issued, submit listing to HMRC. Required: software name (VATflow by United Fintec), description, supported VAT schemes, price (from £2/month), accessibility statement, link to privacy policy. Listed at gov.uk/guidance/find-software-thats-compatible-with-making-tax-digital-for-vat.
