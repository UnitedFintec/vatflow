Accessibility Statement

Our commitment to making VATflow accessible to all users

📅 Last updated: 23 April 2025 📋 WCAG 2.1 Level AA 🏛 United Fintec Limited

This accessibility statement applies to VATflow, operated by United Fintec Limited at vatflow.co.uk. We are committed to making this service accessible to all users, in accordance with the Public Sector Bodies (Websites and Mobile Applications) Accessibility Regulations 2018 and WCAG 2.1 Level AA.

1 Compliance Status

✅ VATflow is fully compliant with WCAG 2.1 Level AA. We have reviewed the service against all applicable success criteria and believe it meets the required standard.

WCAG 2.1 Level AA defines requirements for accessibility of web content. Meeting this standard means the service can be used by people with a wide range of disabilities, including visual, auditory, physical, speech, cognitive, language, learning, and neurological disabilities.

2 Technical Accessibility Features

Visual Accessibility

  • Colour contrast ratios meet or exceed WCAG 2.1 AA minimum (4.5:1 for normal text, 3:1 for large text)
  • Text can be resized up to 200% without loss of content or functionality
  • No information is conveyed by colour alone — icons and text labels accompany all colour-coded elements
  • Focus indicators are visible on all interactive elements
  • The interface is compatible with Windows High Contrast mode

Keyboard Accessibility

  • All functionality is available using a keyboard without requiring a mouse
  • Tab order follows a logical and predictable sequence throughout the application
  • No keyboard traps — users can always navigate away from any component using standard keys
  • Keyboard shortcuts do not conflict with browser or operating system shortcuts

Screen Reader Compatibility

  • All form fields have visible, programmatically associated labels
  • ARIA attributes are used where native HTML semantics are insufficient
  • Error messages are announced to screen readers automatically
  • Page titles are descriptive and unique for each view
  • Images have meaningful alternative text or are marked as decorative where appropriate
  • Compatible with NVDA, JAWS, and VoiceOver screen readers

Cognitive Accessibility

  • Plain English used throughout — no unnecessary jargon
  • Error messages clearly explain what went wrong and how to fix it
  • Important actions require confirmation before proceeding (e.g. submitting a VAT return)
  • Progress indicators show users where they are in multi-step processes
  • Session timeout warnings are displayed with sufficient time to respond

5 How We Tested This Service

VATflow was tested for accessibility using the following methods:

  • Automated testing: WAVE Web Accessibility Evaluation Tool and axe DevTools browser extension
  • Keyboard navigation testing: Full walkthrough of all application screens using keyboard only
  • Screen reader testing: Tested with NVDA on Windows and VoiceOver on macOS
  • Colour contrast checking: All colour combinations verified using the WebAIM Contrast Checker
  • Zoom testing: Application tested at 200% browser zoom
  • Code review: HTML validated against W3C standards; ARIA implementation reviewed manually

Testing was last conducted in April 2025 by United Fintec Limited.
